Clásico de La Plata

Esta plataforma de análisis de encuentros históricos entre Estudiantes y Gimnasia ofrece a los seguidores del fútbol la posibilidad de explorar detalladamente la rica historia de enfrentamientos entre estos equipos. Con funciones de filtrado intuitivas, los usuarios pueden investigar y examinar partidos pasados por fecha, tipo de torneo, equipos involucrados y estadios, brindando una experiencia interactiva para revivir la emocionante rivalidad entre Gimnasia y Estudiantes.

Frases del D1eg0

El proyecto se centra en mostrar citas aleatorias de Diego Armando Maradona en una página web. Se ha implementado una función en JavaScript que selecciona al azar una de las frases célebres del legendario futbolista argentino y la muestra dinámicamente en un elemento HTML. Estas frases representan expresiones icónicas y memorables del legado de Diego.

Moviez

Este proyecto, originalmente parte de ARGENTINA PROGRAMA, se ha personalizado con mejoras y extensiones. Diseñado con JavaScript, Bootstrap y CSS, utiliza la API de The Movie Database (TMDb) como su fuente principal de datos sobre películas. Permite a los usuarios explorar películas populares, las mejor valoradas y los próximos estrenos, así como buscar películas por director o actor. Al seleccionar una película, muestra un modal con detalles esenciales como título, descripción, calificación y año de estreno. Esta interfaz simplificada está orientada a facilitar la exploración y acceso a información relevante sobre películas mediante la integración de la API de TMDb."
